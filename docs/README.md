# GitHub Pages site for *Consensus Change Standards*

This folder is published as a website by GitHub Pages (Settings → Pages → branch `main`, folder `/docs`).

## The interactive scorecard

**Live → https://asaffulks.github.io/consensus-change-standards/scorecard/**

[`scorecard/index.html`](scorecard/index.html) is a self-contained, interactive version of the **20-criterion readiness standard from Section 5.1** of *Consensus Change Standards* (Asaf Fulks, 5th ed. 2026). Score any proposed Bitcoin consensus change in the browser — live Green/Yellow/Orange/Red classification, Taproot and BIP-110 preloaded as reference scorings, copy-to-share output. No build step, no dependencies; open the file locally or visit the live URL.

`index.html` in this folder is just a redirect to `scorecard/` so the bare site URL also lands on the tool.

> GitHub Pages serves this site only from a folder named `docs` — that outer folder can't be renamed without taking the site offline. Anything *inside* it (like `scorecard/`) can be named freely.
